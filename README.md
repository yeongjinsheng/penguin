# Penguin Books Scraper

A Python web scraper that extracts book titles and authors (mostly Epictetus related) from Penguin Books search results using `httpx` and `selectolax`.

## Getting Started

### 1. Prerequisites
Make sure you have Python installed.

### 2. Setup
Clone the repository and set up your virtual environment:

```bash
# Clone the project
git clone [https://github.com/yeongjinsheng/penguin.git](https://github.com/yeongjinsheng/penguin.git)
cd Penguin

# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

### 3. Install requirements.txt
```bash
pip install -r requirements.txt
```
